# 食事バランスノート

食事内容とHealthの数値入力から、今日の推定栄養と明日足したい栄養・食材を表示する静的サイトです。

## Vercelで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. この `vercel-site` フォルダ内のファイルをリポジトリの直下に置きます。
3. Vercelで `Add New...` から `Project` を選びます。
4. 作成したGitHubリポジトリを選択します。
5. Framework Presetは `Other` のままで進めます。
6. Build Commandは空欄、Output Directoryも空欄で公開します。

## 使い方

公開されたURLをスマートフォンで開きます。
iPhoneの場合はSafariの共有ボタンから「ホーム画面に追加」を選ぶと、アプリのように開けます。

## データについて

入力内容は利用者本人の端末内に保存されます。
食事内容やHealthの数値が、この静的サイトの管理者やVercelに送信される処理は入れていません。

## 注意

栄養計算は日常管理の目安です。
診断や治療の代わりにはなりません。
